# HomeScape - Premium Real Estate Website Template

## 📌 ภาพรวมโปรเจค (Project Overview)
**HomeScape** เป็นเทมเพลตเว็บไซต์อสังหาริมทรัพย์ (Real Estate) ที่มีความทันสมัย สวยงาม และพร้อมใช้งานได้ทันที (Production-ready) ถูกออกแบบมาให้มีความพรีเมียมคล้ายกับผลิตภัณฑ์ระดับ SaaS โดยมาพร้อมกับระบบแอนิเมชันที่ลื่นไหล, การจัดตัวพิมพ์ที่สวยงาม, โหมดกลางคืน (Dark/Light Mode), การออกแบบที่รองรับทุกขนาดหน้าจอ (Responsive) และมีหน้า Admin Dashboard สำหรับจัดการระบบหลังบ้านที่ครบครัน

## 🛠️ Tech Stack
- **HTML5**: โครงสร้างหลักของเว็บไซต์และ Semantic Markup
- **Tailwind CSS**: เฟรมเวิร์ก CSS หลักสำหรับการจัดหน้าตาเว็บไซต์ (ใช้งานผ่าน CDN พร้อมกับไฟล์ตั้งค่า `assets/js/tailwind-config.js`)
- **Vanilla JavaScript**: ใช้สำหรับจัดการการโต้ตอบต่างๆ, ระบบ Theme, และ Admin Dashboard (ไม่ต้องพึ่งพา jQuery, React หรือ Vue)
- **Phosphor Icons**: ไอคอนที่ทันสมัยและสวยงาม (ใช้งานผ่าน CDN)
- **Google Fonts**: ฟอนต์ Inter แบบ Modern Typography
- **ApexCharts & FullCalendar**: เลบรารีเสริมสำหรับแสดงกราฟสถิติและปฏิทินในหน้า Admin Dashboard

## 📁 โครงสร้างไฟล์ (File Structure)
โครงสร้างโฟลเดอร์หลักของโปรเจคถูกจัดระเบียบดังนี้:

```text
Home-Scape/
├── index.html            # หน้าแรก (Home Page)
├── listing.html          # หน้ารายการอสังหาริมทรัพย์ (Properties Listing)
├── detail.html           # หน้าดูรายละเอียดอสังหาฯ (Property Detail)
├── about.html            # หน้าเกี่ยวกับบริษัทและทีมงาน (About Us)
├── contact.html          # หน้าติดต่อสอบถาม (Contact)
├── compare.html          # หน้าเปรียบเทียบอสังหาริมทรัพย์ (Property Comparison)
├── email-template.html   # ตัวอย่างเทมเพลตอีเมล (Email Template)
├── README.md             # ไฟล์อธิบายโปรเจคนี้ (ไฟล์นี้)
├── admin/                # ส่วนของระบบหลังบ้าน (Admin Dashboard)
│   ├── index.html        # หน้าหลัก Dashboard แสดงสถิติและภาพรวม
│   ├── login.html        # หน้าเข้าสู่ระบบ (Login)
│   ├── properties.html   # จัดการรายการอสังหาฯ
│   ├── property-form.html# ฟอร์มสำหรับเพิ่ม/แก้ไขข้อมูลอสังหาฯ
│   ├── leads.html        # จัดการผู้สนใจ/ข้อมูลลูกค้า
│   ├── calendar.html     # ปฏิทินการนัดหมายต่างๆ
│   └── settings.html     # การตั้งค่าระบบ
└── assets/               # โฟลเดอร์เก็บไฟล์ Resource ต่างๆ
    ├── css/              # ไฟล์ CSS เสริม (เช่น style.css)
    └── js/               # ไฟล์ JavaScript (เช่น main.js, tailwind-config.js)
```

## ✨ ฟีเจอร์หลัก (Key Features)
- **Modern & Clean Design**: ดีไซน์พรีเมียม สไตล์เรียบหรู ตอบโจทย์ธุรกิจอสังหาริมทรัพย์ระดับไฮเอนด์
- **Fully Responsive**: โครงสร้างแบบ Mobile-first รองรับการแสดงผลบนทุกหน้าจอ (มือถือ แท็บเล็ต เดสก์ท็อป) ได้อย่างสมบูรณ์แบบ
- **Dark/Light Mode**: มีปุ่มสลับธีมสว่างและมืดอย่างสมูท (พร้อมระบบจดจำการตั้งค่าใน Local Storage)
- **Smooth Animations**: เอฟเฟกต์เปิดตัวขณะเลื่อนหน้าจอ (Scroll Reveal) และลูกเล่นการ Hover ที่ทำให้เว็บไซต์ดูมีชีวิตชีวา
- **Admin Dashboard แบบครบวงจร**: มีหน้าจัดการหลังบ้านที่ออกแบบมาอย่างดี ทั้งกราฟวิเคราะห์ข้อมูล (Analytics), ตารางจัดการลูกค้า และปฏิทิน (FullCalendar)
- **Mortgage Calculator**: ระบบคำนวณสินเชื่อบ้านเบื้องต้นในหน้าดูรายละเอียด (`detail.html`)
- **Zero Build Setup Needed**: ไม่ต้องติดตั้ง Node.js หรือใช้คำสั่ง build ใดๆ สามารถเปิดไฟล์เพื่อดูและแก้ไขได้ทันที

## 🚀 การเริ่มใช้งาน (Getting Started)
เนื่องจากโปรเจคนี้ถูกสร้างขึ้นมาในรูปแบบ Static HTML โดยไม่มี Build Step ที่ซับซ้อน คุณสามารถเริ่มใช้งานได้ทันทีผ่านขั้นตอนต่อไปนี้:

1. **ดาวน์โหลดโปรเจค**: โคลน (Clone) หรือดาวน์โหลดไฟล์ทั้งหมดมาไว้ในเครื่องของคุณ
2. **เปิดดูในเบราว์เซอร์**: ดับเบิลคลิกเปิดไฟล์ `index.html` หรือไฟล์หน้าอื่นๆ ด้วยเบราว์เซอร์เพื่อดูผลลัพธ์ได้ทันทีโดยไม่ต้องจำลองเซิร์ฟเวอร์
3. **การแก้ไขเนื้อหา**: ใช้ Code Editor (เช่น VS Code) เพื่อแก้ไขข้อความ รูปภาพ หรือลิงก์ต่างๆ ในไฟล์ HTML ได้โดยตรง
   - หากต้องการเปลี่ยนสีธีมหลัก ปรับขนาด หรือตั้งค่าเกี่ยวกับ Tailwind สามารถเข้าไปแก้ไขได้ที่ `assets/js/tailwind-config.js`
4. **การใช้งานจริง (Deploy)**: อัปโหลดโฟลเดอร์โครงการทั้งหมดขึ้น Web Hosting หรือใช้งานบริการฟรีต่างๆ เช่น Netlify, Vercel, GitHub Pages หรือ Host ทั่วไปได้เลยทันที

## 💡 จุดที่สามารถต่อยอดได้ (Potential Enhancements)
- **เชื่อมต่อกับ Backend/Database**: นำหน้าเว็บและฟอร์มต่างๆ ไปเชื่อมกับฐานข้อมูลและระบบ Backend จริง (เช่น Node.js, Python, PHP) เพื่อบันทึกข้อมูลและดึงข้อมูลอสังหาริมทรัพย์ออกมาแบบไดนามิก
- **ระบบ Authentication**: พัฒนาระบบล็อกอินด้วย JWT หรือ Session ในหน้าเว็บหลักและหน้า `admin/login.html` เพื่อป้องกันการเข้าถึงจากผู้ที่ไม่ได้รับอนุญาต
- **ระบบค้นหาและกรองข้อมูลแบบ Real-time**: นำ Backend เข้ามาช่วยคำนวณผลการค้นหา และทำ Filter ข้อมูลอสังหาฯ ให้ทำงานได้จริง
- **การเชื่อมต่อ API สินเชื่อ**: เปลี่ยนระบบคำนวณสินเชื่อให้เป็นการดึงข้อมูลอัตราดอกเบี้ยจริงจากธนาคารผ่าน API
- **ระบบสลับภาษา (i18n)**: รองรับการแสดงผลหลายภาษาเพื่อขยายฐานลูกค้าชาวต่างชาติ
